Cardwall is a project management and organising tool for Forward3D, currently available to company employees and clients. The app allows users to create 'cards' relating to projects or part of projects, tag them, add/remove users, include due dates, sticky notes, priority levels, colour code them and associate them with clients. An internal messaging system has also been created where users can message each other on a specific card or wall with the ability to add images and attachments. Cards are moved from lane to lane as the project reaches completion, when they are moved to 'Signed Off' and can be archived. The app is built within Forward3D's app management system known as Stage. A full mobile app version of Cardwall is currently in production.
